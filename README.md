# Survival Arena

## Description
Survival Arena is a browser-based survival game where the player controls a blue square and avoids falling enemies. The goal is to survive as long as possible while the game becomes harder over time. The player gains score by surviving and loses lives when colliding with enemies.

## How to Play
- Use the arrow keys to move the player
- Avoid the red enemy blocks
- Survive as long as possible
- The game gets faster as levels increase
- The game ends when all lives are lost

## Controls
- Arrow Up = Move Up
- Arrow Down = Move Down
- Arrow Left = Move Left
- Arrow Right = Move Right

## Features
- Player movement
- Enemy spawning system
- Collision detection
- Score tracking
- Lives system
- Level progression
- Increasing difficulty
- Game over screen

## Technology Used
- HTML
- CSS
- JavaScript
- HTML Canvas
